Промт для доработки Desktop Frontend (Vue 3)
Context: У меня уже реализован рабочий бэкенд и начата реализация фронтенд десктопного интерфейса, уже реализован функционал с Telegram Mini App (TMA). Текущий код содержит логику и компоненты для мобильного отображения. Твоя задача: проанализировать текущую кодовую базу и реализовать полноценный Desktop-интерфейс, не затрагивая и не ломая логику Mini App.

1. Архитектурная задача
Адаптивность: Используй Tailwind CSS для разделения интерфейсов. Мобильная версия (TMA) остается как есть, а для экранов lg: и выше реализуй полноценную админ-панель.

Layout: Реализуй DesktopLayout.vue с левым скрываемым сайдбаром (sidebar).

Сайдбар должен иметь плавную анимацию (transition-all duration-300).

В сайдбаре 4 блока: Организация, Заказы, Клиенты, Продукты (с подразделами из ТЗ).

2. Спецификация разделов (Desktop View)
Проанализируй существующие API-методы и интегрируй их в следующие десктопные компоненты:

Блок Организация:

Создай форму настройки организации. Поля: iiko_organization_id (ID Вашей организации в IIKO) и api_key (API ключ от iiko cloud API).

Таблицы для Терминалов и Типов оплат с кнопками "Загрузить из IIKO".

Меню: Реализуй двухэтапный процесс: кнопка "Выгрузить меню" -> список доступных меню -> выбор и окончательная загрузка.

Блок Заказы:

Полноразмерная таблица заказов.

Действия (Иконки Iconify): eye (открыть модальное окно с составом заказа в таблице), refresh (запрос на проверку статуса в iiko).

Блок Клиенты:

Управление пользователями и адресами через модальные окна.

Связка: Один клиент — несколько адресов доставки.

Блок Продукты:

Таблицы для Продуктов и Модификаторов с фильтрацией.

3. Технические требования к реализации
Icons: Используй @iconify/vue. Все кнопки внутри таблиц должны быть только иконками (semantic icons).

UI Framework: Если в проекте уже используется библиотека (например, Element Plus или PrimeVue), используй её компоненты (Table, Dialog, Drawer).

State: Используй существующие Pinia store для данных, не создавай дублирующую логику.

Безопасность: Доступ к десктопному фронту только после авторизации (Login/Password).
